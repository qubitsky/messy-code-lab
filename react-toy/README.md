## react 是什么

构建前端UI的框架库

为什么说是框架？

所谓框架是指他定义了一种构建UI的模式。系统整体上的约定，或者说模式，可以被称为框架。

UI = Fn(data)

可预测的

什么叫副作用？

除了入参外，额外影响UI的作用

组合优于继承

### 类组件与函数组件

类组件

面向对象编程思想，通过继承复用逻辑
内部结构更清晰

继承灵活性较差
this 指向性很迷惑
业务逻辑分散在生命周期函数中

函数组件

函数式编程思想，输入不变情况下，输出唯一
内部是个黑盒
